如果你不使用Sub—Store，你应该额外开设新的机场订阅策略组，有几个机场开几个策略组，步骤是首页——代理服务器——相应的机场订阅策略组——使用外部代理列表——粘贴到外部代理列表里——保存，然后再到全部节点策略组中，选择包含另一个策略组的策略，把刚刚设置的机场订阅全部打上勾，再点击完成后保存

### 教程
1.使用Sub-Store将你的订阅全部整合到一起，再将整合到一起的订阅链接添加到全部节点策略组，步骤为首页——代理服务器——全部节点——使用外部代理列表——粘贴到外部代理列表里——保存，之后即可使用
